Fast Campus 4/22(DJango)

Created: Apr 22, 2019 7:30 PM
Tags: Django

  1. 새로 시작할 프로젝트 폴더를 원하는 위치에 생성한다.

    -mkdir [폴더명]

  2. 해당 폴더에 가상 환경을 만든다.

    1) pyenv virtualenv [원하는 파이썬 버전] [원하는 가상환경 이름]

    ex) 3.7.2              ex) DJango

    2) pyenv versions ⇒ 해당 명령어로 생성된 가상 환경 확인

    3) 터미널에서 가상 환경을 씌울 폴더로 이동 후

    pyenv local [가상 환경 명] ⇒ 해당 명령어로 원하는 폴더에 가상 환경 덮기

  3. 해당 폴더에서 Django 설치 (현재 2.2가 최신이나 버그가 많아 2.1버전 다운로드)

  • pip install django ⇒ 최상위 버전으로 장고 설치
  • pip uninstall django ⇒ 장고 삭제
  • pip install django==[장고버전] ⇒ 원하는 버전의 장고 설치
  • python -m django –version ⇒ 버전확인
  1. 현재 프로젝트에 사용된 pip 리스트 만들기 (txt 파일)
  • pip freeze > requirements.txt ⇒ 현재까지 사용된 pip 리스트가 적힌 텍스트 파일 생성

  • pip install -r requirements.txt ⇒ 현재 프로젝트 환경과 동일하게 다른 프로젝트를 진행할 시

    해당 파일을 새로 만든 폴더로 이동 후 명령어 입력
  1. 장고 프로젝트 생성

django-admin startproject config .

  1. 서버 가동 ⇒ 가동 해제 명령어 control + c

python manage.py runserver [IP]:[포트 번호] ⇒ IP 와 포트 번호는 안 써도 무관

터미널 창에 떠있는 아이피 주소 클릭하면 mypage 접속 가능
